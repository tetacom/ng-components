# Кнопка

Button (кнопка) — компонент, который призывает пользователя к совершению определенного действия в интерфейсе,
например: открыть следующую страницу, заказать товар или войти в свой аккаунт.

Основной призыв к действию. Предназначены для выполнения какого-либо действия в системе.

### Как использовать

За создание кнопки отвечает компонент

Основные правила применения кнопки:

- Текст кнопки всегда должен быть с заглавной буквы;
- Текст на кнопке должен сообщать пользователю, что произойдёт, если он нажмёт на кнопку (например, Сохранить, Добавить
  и т. п.);
- Не перенасыщать страницу большим количеством Primary кнопок. Не распологать их рядом друг с другом.

### Типы кнопок

В системе существуют всего 3 типа кнопок

- Primary (основная)
- Ghost (без фона)
- Outline (с обводкой)

### Размер кнопок

Есть 3 размера кнопок — s, m и l. В основном в проектах используются кнопки размера m.

### Свойства

| Свойство | Тип                                 | Описание                                                                                                                          |
| -------- | ----------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| viewType | `'rounded' \| 'circle' \| 'brick'`  | Свойство округления компонента                                                                                                    |
| size     | `'s' \| 'm' \| 'l'`                 | Размер кнопки                                                                                                                     |
| disabled | `boolean`                           | Отключает реакцию компонента на клик пользователя                                                                                 |
| palette  | `string`                            | Отвечает за цвет фона кнопки. Принимает название палитры (например: yellow → color-yellow-50). Работает только при view="primary" |
| view     | `'primary' \| 'outline' \| 'ghost'` | Свойство, которое отвечает за тип кнопки                                                                                          |
| square   | `boolean`                           | Отвечает за форму кнопки                                                                                                          |
